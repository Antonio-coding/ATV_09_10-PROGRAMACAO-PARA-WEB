Operações Bancárias

Para realizar operações bancárias, siga estas etapas:

  1. Escolher o Tipo de Conta: Use o menu suspenso "Tipo de Conta" para selecionar o tipo de conta desejado: Conta Corrente, Conta Poupança ou Conta Universitária.
  2. Inserir Saldo Inicial: Digite o saldo inicial desejado na caixa "Saldo". Certifique-se de que o campo esteja preenchido corretamente e seja um valor numérico válido.
  3. Inserir: Clique no botão "Inserir" para criar uma nova conta bancária com os detalhes fornecidos. Você pode criar várias contas escolhendo diferentes tipos e saldos.
  4. Deletar: Selecione uma conta da lista que deseja excluir. Clique no botão "Deletar" para remover a conta selecionada. A conta será removida permanentemente.
  5. Visualizar Extrato: Clique no botão "Visualizar" para ver o extrato de todas as contas criadas. O extrato exibirá informações sobre cada conta, incluindo tipo e saldo.
